# Szybkie potęgowanie

```

123**7 = 1 * 123 * 123 * 123 * 123 * 123 * 123 * 123
                                   łącznie: 7 mnożeń

123**7000 = 1 * 123 * 123 * ... * 123
                                łącznie: 7000 mnożeń

123**7000 = 

123*123              = 123 ** 2 = 15129
15129 **2            = 123 ** 4
228886641 ** 2       = 123 ** 8
                     = 123 ** 16
                     = 123 ** 32
                     = 123 ** 64
                     = 123 ** 128
                     = 123 ** 256
                     = 123 ** 512
                     = 123 ** 1024
                     = 123 ** 2048
                     = 123 ** 4096    (12 mnożeń, zamiast 4096)
                     = 123 ** 4096 * 123 ** 2048 = 123**6144
                     = 123 ** 6144 * 123 ** 512 = 123 ** 6656
                     = 123 ** 6656 * 123 ** 256 
                     itd.
```

```
7000 = 4096 + 2048 + 512 + 256 + 64 + 16 + 8

7000 = 2**12 + 2**11 + 2**9 + 2**8 + 2**6 + 2**4 + 2**3

```
